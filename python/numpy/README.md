## Projects Included

| File                      | Description                                                |
|---------------------------|------------------------------------------------------------|
| `diffusion.py`            | 2D diffusion simulation using finite differences.          |
| `wave_equation.py`        | 2D damped wave equation solver                             |
| `euler_gas_equation.py`   | Euler equations for compressible gas dynamics in 2D.       |
| `animation_helpers.py`    | Utility functions to save and create GIF animations.       |
| `some_other_sim.py`       | Additional physics simulation example (custom).            |
